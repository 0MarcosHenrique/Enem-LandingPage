## 🧠 About the Project

This landing page was created with the goal of maximizing conversions through:

* AIDA structure (Attention, Interest, Desire, Action)
* Psychological triggers (scarcity, urgency, social proof)
* Premium design focused on user experience
* Integration with external checkout (Kiwify)
* Use of **Artificial Intelligence as support for development and ideation**

---

## 🤖 AI Usage

This project leveraged Artificial Intelligence tools for:

* Copywriting generation and refinement
* Landing page structuring
* Design and UX suggestions
* Code optimization

> AI was used as a support tool, while implementation, adjustments, and customization were done manually.
